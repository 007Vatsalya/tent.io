Profile Info Types

Core

https://tent.io/types/info/core/v0.1.0

Property Required Type Description
entity Optional String The canonical entity identitifier.
licenses Optional Array The licenses the entity publishes content under.
servers Optional Array The canonical API roots that can be used to interact with the entity.

Basic

https://tent.io/types/info/basic/v0.1.0

Property Required Type Description
name Optional String Name to be displayed publicly.
avatar_url Optional String URL to avatar to be displayed publicly.
birthdate Optional String Date of birth in one of these formats: YYYY-MM-DD, YYYY-MM, MM-DD
location Optional String Location to be displayed publicly.
gender Optional String Gender to be displayed publicly.
bio Optional String Biography/self-description to be displayed publicly.
